NULL Privacy Policy

Last Updated: 2026

Overview

NULL is a private developer analysis engine.

This policy explains how information may be processed while using NULL.

Information Processed

NULL may process:

- Commands submitted by users.
- Error messages, logs, stack traces, and code snippets provided by users.
- Basic Discord information required for command execution.

AI Processing

User-provided content may be sent to third-party AI providers to generate explanations and analysis results.

Examples include:

- Programming questions.
- Error analysis requests.
- Code explanations.

Data Storage

NULL does not intentionally store user content permanently unless required for system functionality, debugging, security, or administration.

Security

Reasonable measures are taken to protect processed information. However, no system can guarantee absolute security.

Third-Party Services

NULL may rely on third-party providers including:

- Discord
- AI service providers
- Hosting providers

Use of these services is subject to their own policies.

Contact

For questions regarding this policy, contact the maintainers of NULL.

Changes

This policy may be updated at any time without prior notice.
